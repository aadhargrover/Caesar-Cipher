What Are Ciphers?

Ciphers are methods to transform readable text (plaintext) into an unreadable format (ciphertext) using rules or keys. They protect confidentiality and lay the foundations for modern cryptography.

On this site you’ll find interactive tools and concise explanations for several classic ciphers. They’re great for learning how substitution and transposition work before diving into modern crypto.

How These Ciphers Work

    

      

        
Caesar Cipher

        

         The Caesar cipher is an ancient substitution cipher that shifts each letter of the alphabet by a fixed number of positions. It was widely used in Roman times for secret messages, but due to its simplicity, it is highly vulnerable to modern cryptanalysis.
        

      

      

        
Playfair cipher

        

          The Playfair cipher is a digraph substitution method that encrypts letter pairs using a 5×5 key square. It replaces pairs based on rules for rows, columns, and rectangles, making it stronger than simple monoalphabetic ciphers but still vulnerable to frequency analysis.
        

      

      

        
Vigenère cipher

        

          The Vigenère cipher is a polyalphabetic substitution method that uses a repeating keyword to shift letters by varying amounts. This makes patterns less obvious than in simple ciphers like Caesar, giving it more security, though it can still be cracked with frequency analysis.
        

      


      

        
Pigpen Cipher

        

        The Pigpen cipher is a geometric substitution cipher that replaces letters with symbols derived from grids or shapes. Popular with secret societies like the Freemasons, it looks mysterious but offers little real security since the symbols can be memorized and cracked easily.        
        

      


      

        
Atbash cipher

        

          The Atbash cipher is a simple monoalphabetic substitution method where each letter is replaced by its opposite in the alphabet, such as A becoming Z and B becoming Y. Though historically significant, it provides no real security since the pattern is fixed and easy to decode.
        

      


      

        
ROT13 cipher

        

          The ROT13 cipher is a simple substitution method that shifts each letter 13 places in the alphabet. Because applying it twice returns the original text, it is often used for obfuscation rather than true encryption, providing no real cryptographic security.
